# Object Detection


## Introduction

This project is a simple object detection project using the Detectron2 library.


## Tools

1. FastAPI (for API)
2. Celery (for async tasks)
3. Redis (for message broker)
4. Docker (for containerization)


## Usage

1. Clone the repository
2. ```bash
   docker compose up
   ```
